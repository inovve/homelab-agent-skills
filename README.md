# Agent Skills

A collection of [Agent Skills](https://agentskills.io) for AI agents like Claude Code.

| Skill | Description |
|---|---|
| [outline](outline/) | Full read/write access to an [Outline](https://www.getoutline.com) wiki (cloud or self-hosted) via its REST API — search, documents, collections, comments, attachments, shares, exports, and more. |

Each skill folder contains its own README with setup instructions. Credentials are never stored in the skills — see each skill's configuration section.
